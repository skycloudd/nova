# If/Else Statements

If/Else statements are used to control the flow of a program. They're used to run different code depending on the value of a boolean expression.

## Syntax

```swift
if <expression> then
    <code>
else
    <code>
end
```

The `else` block is optional.

## Example

```swift
if true then
    // ...
end
```

```swift
if 2 + 2 == 5 then
    action print: 4;
else
    action print: true;
end
```
